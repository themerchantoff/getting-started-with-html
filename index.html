The Basics of the Web

The web is made up of lots of HTML documents, HTML stands for Hyper Text Markup Languge. So we can difine that a web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.The web is a bunch of computers that communicate with each other. When a person goes to a web page like www.google.com, their computer sends a HTTP Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.This page is a grate way to start.

Basic Programing

There are 6 important tags that "every" programer should know:
✤b tag --> Makes "contents" bold.
✤em tag --> Makes "contents" italic.
✤br tag --> Which is a inline, and makes space betwen lines.
✤p tag --> Whis is a block makes diffrent paragrahs.
✤a tag --> Makes text into a hyperlink(ex.-"www.google.com")
✤image tag --> Links an imaage to your HTML docuemt.

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.
